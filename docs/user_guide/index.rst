Core Concepts
-------------

Siamese Prototype is a logic programming-based rule engine that uses the following core concepts:

* **Facts**: Statements known to be true
* **Rules**: Logical rules that derive new knowledge based on conditions
* **Queries**: Questions posed to the knowledge base
* **Variables**: Placeholders representing unknown values in rules and queries
* **Unification**: The process of matching and binding variables
